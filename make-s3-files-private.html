Making S3 files private in bulk

Sun 16 December 2018

Have you ever needed to make all files in a certain AWS S3 bucket private? Here is a short script, using GNU parallel and aws-cli, to do just that (replace BUCKET_NAME with the actual name of the bucket):

1
2
3
4
5
aws s3 ls --recursive s3://BUCKET_NAME | \
    cut -d' ' -f5- | \
    awk '{print $NF}' | \
    parallel --lb \
    'aws s3api put-object-acl --acl private --bucket BUCKET_NAME --key "{}"; echo "{}"'
